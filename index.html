Rules

1. When Value is 0. Only addition Operation Allowed.
2. When value is greater then 10000. No Operation Allowed.
3. For DIVISION & MULTIPLICATION - the input value must be larger than 0.
4. DIVISION is allowed when count value is greater than input value.
5. Input value must be an integer number. Negative numbers are not allowed.
